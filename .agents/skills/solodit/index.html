Solodit Skill

Search smart contract vulnerabilities with Claude Code

Access 50,000+ security findings from top audit firms. Filter by severity, tags, protocols, and more.

About

Solodit is the world's largest database of smart contract security vulnerabilities, aggregating findings from Cyfrin, Sherlock, Code4rena, Trail of Bits, OpenZeppelin, and more.

This skill gives AI agents direct access to search and analyze these findings programmatically.

Features

  • Search by keywords (reentrancy, oracle, flash loan)
  • Filter by severity: HIGH, MEDIUM, LOW, GAS
  • Filter by audit firm, protocol, or category
  • Sort by recency, quality score, or rarity
  • Full markdown content with code examples
  • Rate limit aware with automatic warnings

Installation

$ npx skills add BowTiedSwan/solodit-api-skill
Or with curl: curl -fsSL https://raw.githubusercontent.com/BowTiedSwan/solodit-api-skill/main/install.sh | bash

Requires a free API key from solodit.cyfrin.io
